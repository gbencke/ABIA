1. Instalação do Sistema

Os passos abaixo devem ser executados para efetuar a instalação do sistema

1.1 Instalação do Cygwin e PostgreSQL

É necessário instalar o Cygwin, que é um emulador de Linux para Windows. Ele pode ser obtido em sources.redhat.com. É importante que l PostgreSQL seja instalado automaticamente pelo Cygwin como demonstrado abaixo:



Após a instalação, é necessário criar um database chamado ABIA, cheque a documentação do PostgreSQL. Crie uma aplicação Web com o nome ABIA.

1.2 Instalação do JDK

É necessário instalar o JDK 1.4 que pode ser obtido em http://java.sun.com/

1.3 Instalação do Apache Tomcat 4.21

É necessário instalar o Apache Tomcat 4.21 que pode ser obtido em http://jakarta.apache.org/

1.4 Instalação do Software

Para instalar o software, baixe o pacote de http://abia.sourceforge.net/, e copie o diretório test para algum diretório da sua máquina. Não é necessário executar nenhum software de instalação.
