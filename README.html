1. Instalação e Execução do Programa

 O Documento de Instalacao contêm todas as informações para a instalação e execução do programa.

2. Estrutura e Organização dos Fontes e Dados de Teste

Os fontes estão organizados nos diretorios abaixo:

2.1 Diretório Downloaded

Esse diretorio contêm os dados de teste previamente "Garimpados" na Internet. Os arquivos são os seguintes:
_cotacoes*.txtEsses arquivos contêm séries históricas obtidas no Yahoo! Finance
200*.txtEsses arquivos são as séries históricas acima, porém reescritas no formato BDI da BOVESPA
ListaEmpresas*.txtLista de empresas negociadas na BOVESPA
DiretoriosLista de balancetes emitidos na CVM, por data de publicação

2.2 Diretório org

Fontes em Java do Container de Agentes e classes compiladas

2.3 Diretório test

Diretório de Teste do Container de Agentes. O aplicativo tem que ter os seguintes arquivos e diretorios:s
agentsArquivos JAR dos agentes utilizados pelo Container de Agentes
binArquivos JAR principais do Container de Agentes (ABIA.JAR) e CLIPServer
confArquivos de configuração e regras dos agentes do sistema

2.3 Diretório utils

Aplicativos utilitarios utilizados pelo Container de Agentes.
ChartGeneratorPrograma em VB utilizado para gerar Graficos em Excel
CLIPServerAdaptação do CLIPS para aceitar comandos remotos
libsBibliotecas JAR auxiliares
SimulatorSimulador de Sites Financeiros
